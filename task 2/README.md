# TechFest 2026 - Event Landing Page

Simple and responsive event landing page built with Tailwind CSS.

## Features
- Responsive design
- Event schedule
- Registration form
- Clean UI

## Deploy to GitHub Pages

1. Create repository on GitHub
2. Push code: `git init`, `git add .`, `git commit -m "Initial commit"`
3. Connect: `git remote add origin YOUR-REPO-URL`
4. Push: `git push -u origin main`
5. Enable Pages in Settings > Pages > Select "main" branch
6. Live at: `https://yourusername.github.io/repo-name/`

- HTML5
